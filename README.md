# B&CG - Ball and Coin Game
## Установка - Installation
### RU
Устанавливаем "python", "python-pip" и пишем в терминале:
```
pip install pygame
```
Затем запустите main.py

#### EN
Install "python", "python-pip" and write in the terminal:
```
pip install pygame
```
Then run main.py
